# Kisan Alert User Flow

## Onboarding Flow

Splash
→ Language Selection
→ Login (OTP)
→ Permissions
→ Dashboard

---

## Dashboard Navigation

Dashboard
├── 🌱 What Should I Grow?
├── 📷 Check My Crop
├── 🎤 Ask Kisan AI
├── 🔔 Alerts
└── 👤 Profile

Dashboard also shows:
- Current weather summary
- Farm health score
- Today's AI advice

---

## Crop Recommendation Flow

Dashboard
→ What Should I Grow
→ Step 1: Detect Location
→ Step 2: Soil Scan
→ Step 3: Select Season
→ Step 4: Analyzing
→ Recommendation Result

Purpose:
Recommend the best crop based on:
- Location
- Soil image
- Season

---

## Crop Disease Flow

Dashboard
→ Check My Crop
→ Step 1: Capture Crop Photo
→ Step 2: Analyzing
→ Step 3: Disease Result
→ Step 4: Treatment Advice

Purpose:
Detect crop disease and provide treatment advice.

---

## Ask Kisan AI Flow

Dashboard
→ Ask AI
→ Voice Input / Text Input
→ AI Response
→ Voice Playback
→ Suggested Questions

Purpose:
Allow farmers to ask farming-related questions in their own language.

---

## Alerts Flow

Dashboard
→ Alerts

Includes:
- Weather warnings
- Rain alerts
- Heatwave alerts
- Disease outbreak alerts
- Irrigation advice
- Emergency actions

Purpose:
Provide proactive farm alerts and daily advisories.

---

## Profile Flow

Dashboard
→ Profile
→ Edit Profile
→ Language Settings
→ Notification Settings
→ Linked Family Member
→ Emergency Contacts

Purpose:
Manage farmer account and preferences.
